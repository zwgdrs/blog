# blog

 前端: react+redux+react-router
 
 后端: node+express+mongoDB提供的API
 
 前后端完全分离

### 开始 
修改mondoDb.sh路径为自己本机上的mongo路径

### 运行
    git clone https://github.com/zwgdrs/blog.git
    npm install
    启动mongo：  ./mongoDb.sh
    启动: npm start

##### 开发环境
    浏览器输入 localhost:7070

##### 生产环境
    浏览器输入 localhost:8080
